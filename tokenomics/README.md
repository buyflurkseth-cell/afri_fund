# Tokenomics — Overview

AfriFUND (AFR) is a **fixed-supply token on Solana (SPL)** with a total issuance
of **10,000,000,000 AFR**. This page provides a high-level breakdown of the
allocation — including presale, referral pool, ambassador bonuses, liquidity,
development fund, and team. All figures are fixed at the protocol level and
verifiable on-chain.

Funds raised build real infrastructure, and **100% of project profits are
distributed quarterly** to holders.
