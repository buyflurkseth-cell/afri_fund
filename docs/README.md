# Introduction

**AfriFUND (AFR)** is a public investment token on **Solana** that channels
community and government capital into real East-African infrastructure —
solar farms, roads and water systems — and returns project profits to holders.

* **Network / Standard:** Solana · SPL
* **Total supply:** 10,000,000,000 AFR
* **Profit distribution:** 100% of project profits, quarterly, pro-rata
* **Audit:** SolidProof
