# FAQ

**What is AFR?**
A public investment token on Solana (SPL) that funds real infrastructure and
shares project profits with holders.

**How do I earn?**
Hold AFR for quarterly dividends, and/or join the Ambassador Program to earn
referral commissions across multiple levels.

**What is the minimum purchase?**
0.033 SOL (~$5).

**Is the contract audited?**
Yes — audited by SolidProof and verified on Solscan.

**When are dividends paid?**
Quarterly, via smart contract, directly to holder wallets.
